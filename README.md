# zimenglyu
My personal website : https://zimenglyu.com

# running the site:

1) Install Jekyll
```
sudo gem install bundler jekyll
```

2) Clone the repo
```
git clone https://github.com/cathybgyz/zimenglyu.git
```

3) Run the site
```
cd zimenglyu
bundle exec jekyll serve
```

Then navigate to `http://localhost:4000`
